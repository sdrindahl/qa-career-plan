60–90 Day Roadmap

A structured approach to landing your AI-assisted QA role

Weeks 1–2: Foundation

  • AI fundamentals course (10-15 hours)
  • JavaScript basics (variables, loops, functions)
  • Set up GitHub account & learn git basics
  • Create first GitHub repository

Weeks 3–6: First AI + QA Project

  • Pick one project idea (test case generator recommended)
  • Use AI heavily in development (ChatGPT, Claude)
  • Build & test thoroughly
  • Publish to GitHub with comprehensive README
  • Document your learning process

Weeks 6–10: Second Project + Job Search

  • Build second project (different domain)
  • Update resume with AI + QA focus
  • Refresh LinkedIn profile with new projects
  • Apply for "AI-assisted QA" roles
  • Network in QA/AI communities

Week 10+: Career Direction

  • Target roles: QA Automation Engineer (AI-assisted)
  • SDET with AI workflows
  • AI-assisted QA Engineer
  • Consider 3rd project for competitive edge
  • Continue learning AI advancements

Pro Tip: Keep your GitHub active, update your profile weekly, and share your learning journey!
