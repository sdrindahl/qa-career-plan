60–90 Day Roadmap

A structured approach to landing your AI-assisted QA role

Weeks 1–2: Foundation

  • AI fundamentals course (10-15 hours)
  • JavaScript basics (variables, loops, functions)
  • Set up GitHub account & learn git basics

Weeks 3–6: First AI + QA Project

  • Pick one project idea
  • Use AI heavily in development
  • Build & test thoroughly
  • Publish to GitHub with README

Weeks 6–10: Second Project + Job Search

  • Build second project (different domain)
  • Update resume with AI + QA focus
  • Refresh LinkedIn profile
  • Apply for "AI-assisted QA" roles

Week 10+: Career Direction

  • QA Automation Engineer (AI‑assisted)
  • SDET with AI workflows
  • AI‑assisted QA Engineer
  • Build optional 3rd project for competitive edge

Tip: Keep your GitHub active, update your profile weekly
